基本的な使い方

ナビゲーション

上部のアドレスバーにURLを入力してWebサイトを閲覧。戻る・進む・再読み込みボタンも利用可能。

設定

右上のメニューボタンから設定画面を開き、検索エンジンやテーマカラーを変更できます。

ブックマーク追加

アドレスバー横の星ボタンで、現在のページをお気に入りに追加。

コントリビュート（貢献）

IFoxerはオープンソースプロジェクトです。バグ報告・機能提案・プルリクエストなど、どなたでも歓迎します！

  1. 改善したい点があれば、Issueを立ててディスカッション
  2. リポジトリをForkし、git checkout -b feature/新機能名 でブランチ作成
  3. 変更をコミット（例：git commit -m 'Add AmazingFeature'）
  4. GitHubにプッシュし、Pull Requestを作成

よくある質問（FAQ）

インストール時にウイルス警告が出ます

一部のセキュリティソフトで誤検知される場合がありますが、公式配布ファイルをご利用ください。

7zファイルが開けません

7-Zipなどの解凍ソフトをインストールしてください。

動作環境は？

Windows 10以降を推奨します。

サポートはありますか？

現在はコミュニティベースのサポートを提供しています。ご質問は公式サイトのお問い合わせフォームからどうぞ。
